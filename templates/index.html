SoloBlogger Backend Project URLs

Notice: Use Postman for API Testing

We highly recommend using Postman for API testing. Postman offers a user-friendly interface and powerful features to streamline the process of developing, testing, and debugging APIs.

  • /api-auth/register/

    Please register at the link to add blogs.To register you need to provide the following :-

    1. username
    2. email
    3. password
    4. password1
    You will get a authentication token use it for sending post request for creating a blog, updating a blog, deleting a blog and writing reviews.

  • /api-auth/login/

    Login at the link to add blogs.You will get a token use it for sending post request for creating a blog,updating a blog,deleting a blog and writing reviews.provide the following:-

    1. username
    2. password
    in order to receive the token.

  • api-auth/logout/

    Logout using the token.

  • api/create_blog/

    Only authenticated users can create a blog.Use the token received at the time of registration.

  • /api/update_blog/id

    To update blog you need to pass id of the blog in Url. NOTE :- The Blog Owner Can Edit His/Her Blog.

  • /api/search_blog/

    Pass the parameter(Blog-Title) to search the blog.

  • api/review/id/create/

    Only Authenticated User Can add the review. Add the Id of blog in the url to add the review

  • api/review/id/list/

    Add the Id of blog in the Url to read the reviews of the blog

  • api/review/id/

    Add the Id of Review in the Url to Update the reviews for the blog.NOTE :- The Review Owner Can Edit His/Her Review.
